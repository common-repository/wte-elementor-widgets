== Description ==

The WP Travel Engine – Elementor Widgets provides 20+ Elementor widgets to create travel and tour booking agency websites using [WP Travel Engine](https://wptravelengine.com/?utm_source=wp.org&utm_medium=wte-elementor-widgets&utm_campaign=wp_profile) and Elementor.

WP Travel Engine Elementor Widgets allows you to create and modify travel booking websites with a visual touch, and it also comes with pre-designed templates to streamline your workflow.

If you are a travel entrepreneur, travel web developer or travel theme developer, you will find this plugin helpful. It brings together the two popular plugins, WP Travel Engine and Elementor, to create feature-rich travel websites without writing any code.

### 30 FREE ELEMENTOR WIDGETS AND COUNTING
1. **_Trips_** - Display trips dynamically based on conditions (featured, latest, on sales and custom selection), with five layout options and numerous filters to choose from.
2. **_Destinations_** - Display your destinations with three distinct layouts and numerous filtering options.
3. **_Trip Types_** - Highlight different trip types using three layouts and numerous filtering options.
4. **_Activities_** - Display your activities with three distinct layouts and numerous filtering options.
5. **_Trip Search_** - Display the search filter in either horizontal or vertical orientation, and enhance it with search criteria like destination, duration, price, and activities.
6. **_Advanced Trips 1_** - Showcase tour packages in stunning masonry layouts with the image and trip information (title, location, price, duration etc.). Choose from 4 diverse layouts to match your unique style.
7. **_Advanced Trips 2_** - Showcase tour packages in masonry layouts, with the image and trip information displayed but with bottom pricing placement.
8. **_Advanced Trips 3_** - Showcase your tour packages in a visually appealing grid format on any Elementor page. Filter by latest, featured, or specific categories.
9. **_Advanced Trips 4_** - Showcase your tour packages in a grid format with the image and trip information (title, price, duration, etc.) placed within the image.
10. **_Trips Accordion_** - Display multiple trips in an interactive vertical accordion. Hover on the card, and it will automatically expand to show the content inside.
11. **_Trips Module 1_** - Highlight your desired multiple trips in visually appealing grid layouts. This widget comes with 2 predefined layouts.
12. **_Trips Module 2_** - Showcase your trip in three distinct layouts. Choose from grid layouts with a mix of horizontal and vertical placements, or select a rectangle or circle for your thumbnail images.
13. **_Trips Module 3_** - Showcase your trips in two pre-designed tiles. ​​Place one trip in the spotlight with a larger image to capture your readers’ attention instantly.
14. **_Trips List_** - Present your trips in style with customizable titles, images, and categories. This widget offers multiple layouts to perfectly fit your homepage or category pages.
15. **_Featured Trip_** - Display a featured trip with prominent discounts, destination information and booking details in a visually appealing card format.
16. **_Trips Slider_** - Showcase your tour packages on any Elementor page with an engaging slider layout. Enjoy full control over all settings for a seamless display experience.
17. **_Trips Slider 2_** - Display trips in a carousel format with a prominent button or icon at the bottom to encourage visitors to take action.
18. **_Trips Slider 3_** - Display the trip's featured image with the sleek "cover flow" slider effect, perfect for showcasing your tour packages.
19. **_Trips Tab_** - Create visually appealing tabs to showcase your trip packages in an interactive slider format. Organize the tabs by destination, activity, or trip type for easy browsing.

20. **_Trip Booking_** - Showcase essential trip details, including price, discounts, highlights, availability button and enquiry forms.
21. **_Trip Breadcrumbs_** - Guide users seamlessly through your website using intuitive trip breadcrumbs.
22. **_Trip Carousel_** - Display multiple tour images attractively in a carousel format.
23. **_Trip Cost Excludes_** - Display the trip exclusions using customizable icons and styles.

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select wte-elementor-widgets.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `wte-elementor-widgets.zip`
2. Extract the `wte-elementor-widgets` directory to your computer
3. Upload the `wte-elementor-widgets` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

== FAQs ==

= Can I use the plugin without WP Travel Engine? =

No, you cannot use the WP Travel Engine - Elementor Widgets plugin without WP Travel Engine. The plugin is designed to work with WP Travel Engine and requires the core plugin to be installed and activated.

= Does it work with any theme? =

Yes, the WP Travel Engine - Elementor Widgets plugin is compatible with most WordPress themes. However, a few themes that have been heavily modified might not be compatible.

= Is WP Travel Engine - Elementor Widgets plugin, fully free? =

Yes, the WP Travel Engine - Elementor Widgets plugin is fully free.

= What are the requirements to use WP Travel Engine - Elementor Widgets plugin? =

To use this plugin, you need to have the latest version of WordPress, the WP Travel Engine plugin and the Elementor plugin installed and activated.

= Will it slow down my site speed? =

The plugin is optimized for speed and performance, so it will not slow down your site speed.
